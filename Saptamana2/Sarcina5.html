Notiuni geometrice

Punct:
Punctul este o entitate geometrică fără dimensiuni, reprezentată printr-o poziție unică în spațiu.
Linie:
Linia este o succesiune infinită de puncte dispuse într-o direcție continuă. Nu are lățime sau adâncime și este definită de două puncte distincte.
Plan:
Planul este o suprafață plană, fără grosime, care se întinde la infinit în toate direcțiile. Este definit de cel puțin trei puncte nealiniante.
Poligon:
Un poligon este o figură geometrică bidimensională închisă, formată dintr-o secvență de segmente de linie care se unesc la capete. Exemple de poligoane includ triunghiurile, pătratele și hexagoanele.
Cerc:
Cercul este o figură geometrică bidimensională formată de toate punctele care sunt la o distanță egală față de un punct fix numit centru. Distanța de la centrul la oricare punct de pe cerc se numește rază.
